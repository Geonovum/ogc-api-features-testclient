Demo WFS 3.0 BGT (deelgebied Swifterbant)

Tijdens de werkweek WFS 3.0 (https://geoforum.nl/t/1e-week-juni-wfs-3-0-werkweek/2649/29) heeft PDOK in samenwerking met Geonovum een WFS 3.0 (OGC API Features) ingericht met achterliggende BGT data van Swifterbant. De WFS heeft de status demo (test) en wordt niet bijgewerkt of verrijkt met meer data van andere gebieden.

De demo WFS 3.0 wordt in twee projecties aangeboden, WGS84 (de standaard voor WFS3) en in het RijksDriehoekstelsel (RD, EPSG:28992). De demo-client ondersteunt op dit moment alleen WGS84.

Via deze demo-client kunt u de data via de API verkennen. Het is een basale client, om de werking van de API te illustreren. In het log scherm onder de kaart is te zien hoe de API aangeroepen wordt.
