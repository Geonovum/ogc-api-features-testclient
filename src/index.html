OGC API Features and API Records test client

This client is very basic. The goal is to demonstrate the usage of OGC API Features and API Records implementations and to test different implementations of these API specification. Enter an OpenAPI document URL to see what data is offered by the API or choose one from the list. The logging screen below the map shows which HTTP requests are sent.
